# ビンゴ抽選機

学生団体のイベントで使えるシンプルなビンゴ抽選アプリです。  
1〜75の数字をランダムに1つずつ抽選し、当選済みの番号を画面下部のパネルで管理できます。

---

## 使い方

### 1. ダウンロード

画面右上の緑色の「Code」ボタン → 「Download ZIP」をクリックして、ZIPファイルをダウンロードしてください。  
ダウンロードしたZIPファイルを解凍します。

### 2. 起動

解凍したフォルダの中にある `bingo` フォルダを開き、`index.html` をダブルクリックしてブラウザで開いてください。

> **動作確認済みブラウザ:** Google Chrome / Microsoft Edge / Safari

### 3. 操作方法

| 操作 | 内容 |
|------|------|
| `SPIN` ボタンを押す | 数字がランダムに高速で表示されます |
| `STOP` ボタンを押す | 数字が確定し、当選番号として表示されます |

- 当選した番号は画面下部のパネルに水色でハイライトされます。
- 1〜75の全番号を抽選し終えると、リセットを促すメッセージが表示されます。

### 4. リセット方法

ページをリロード（F5キー、またはブラウザの更新ボタン）すると最初の状態に戻ります。

---

## 注意事項

- インターネット接続は不要です。
- ページをリロードすると、それまでの抽選履歴はすべてリセットされます。
- 音が出ます。会場の環境に応じて、PCの音量を調整してください。
